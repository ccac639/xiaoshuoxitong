# WinkNovel UI 仿制版

基于 WinkNovel 原版软件的 UI 层仿制，包含所有主要功能界面。

## 功能特性

- 🏠 **主页** - 功能介绍与快速入口
- 📚 **书架** - 小说项目管理
- 📖 **小说详情** - 章节管理
- ✍️ **章节编辑器** - 文本编辑
- ✨ **润色** - 原文与润色稿对比
- 📝 **续写** - AI 辅助续写
- 🤖 **AI 配置** - 模型与 API 配置
- 📋 **提示词** - 自定义 AI 提示词
- 📊 **请求记录** - API 调用历史

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   # Python
   python -m http.server 8080
   
   # Node.js
   npx serve
   ```
3. 浏览器访问 `http://localhost:8080`

## 技术栈

- 纯 HTML / CSS / JavaScript
- 无需构建工具
- 本地存储（localStorage）持久化数据

## 主题切换

点击右上角🌙/☀️按钮切换深色/浅色主题。

## 数据存储

所有数据存储在浏览器 localStorage 中，刷新页面不会丢失。

## 与原版差异

- 使用模拟数据，未连接真实后端
- AI 功能为 UI 演示，未实现真实 API 调用
- 部分高级功能简化为 UI 展示
